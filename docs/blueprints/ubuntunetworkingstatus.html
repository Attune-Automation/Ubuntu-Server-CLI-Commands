Step 1 - Get the IP address of all interfaces

Query the status of network links.

networkctl may be used to introspect the state of the network links as seen by systemmd-networkd. Please refer to systemd-network.service for an introduction to the basic concepts, functionality, and configuration syntax.

Connect via ssh:

ssh {ubuntuuser}@{ubuntunode}

Execute the following script: 

                
networkctl status

Step 2 - Display all IP addresses of the host

Hostname is used to display the system's DNS name, and to display or set its hostname or NIS domain name.

Execute the following script: 

                
hostname -I

Step 3 - Show routes

Show routing, network devices, interfaces, and tunnels.

Execute the following script: 

                
ip route
